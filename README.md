# Hospital WebSite (Web Project)

A **Hospital WebSite** web application designed to manage hospital operations efficiently. This project helps in handling patient information, doctor details, appointments, and basic hospital services through a simple and user‑friendly interface.

---

## Project Overview

The Hospital WebSite is developed as an academic / learning project to understand **frontend web development concepts**. It focuses on organizing hospital-related data and improving workflow using a clean UI.

---

## Features

*  Doctor information management
*  Patient details display
*  Appointment booking page
*  Contact & inquiry page
*  Hospital services overview
*  Responsive design (Mobile & Desktop)
*  Simple and clean UI

---

## Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & Layout
* **JavaScript** – Basic interactivity
* **Git & GitHub** – Version control
